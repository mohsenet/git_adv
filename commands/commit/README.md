### Types of commit

```bash
git commit
```

```bash
git commit -m "YOUR_COMMIT"
```

```bash
git commit -m "TITLE" -m "CONTENT"
```


### Set Your Default Editor

```bash
# vi
git config --global core.editor "vi"

# nano
git config --global core.editor "nano"

# VS Code
git config --global core.editor "code --wait"
```
